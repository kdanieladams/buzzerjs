# Buzzer.js Client

Simple buzzer app for debates.  This is the client-side app, written in Vue.js.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
